# Website_Rekomendasi_Tempat_Wisata_Lampung
Project UAS  Web Framwork: Website Rekomendasi Tempat Wisata Lampung berbasis Express.js, Node.js, dan Mongo.DB.

# Deskripsi Proyek
Website rekomendasi tempat wisata di Provinsi Lampung yang dibangun menggunakan Express.js, Node.js, dan MongoDB. Website ini membantu pengguna memperoleh informasi mengenai destinasi wisata serta menyediakan dashboard admin untuk mengelola data wisata.

# Fitur
- Login Admin
- Dashboard Admin
- CRUD Data Wisata
- Detail Wisata
- Daftar Destinasi Wisata

#Teknologi yang Digunakan
- Node.js
- Express.js
- MongoDB
- EJS
- HTML
- CSS
- JavaScript

#Cara Menjalankan Aplikasi

1. Install dependency

npm install

2. Jalankan aplikasi

node app.js

3. Buka browser dan akses localhost yang digunakan.
